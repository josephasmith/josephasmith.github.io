If you're new to Azure DevOps, or need a playground in your hobby time to level up your DevOps skills - you can achieve this easily with a Microsoft account and registering for DevOps.

## What you get

* Access to boards, pipelines, repos
* Unlimited repositories
* 5 Users
* 2GB free artifact storage
* Self-hosted pipelines: Unlimited (max 1 agent)
* MS-hosted: 1800 free minutes/month

The only notable omission is Test Plans, sadly which you would need to pay $52 per month per user (at time of writing) for. Still, there's plenty of free resource included in the Basic plan to get going with. So let's crack on and get set up.

## Registration

Assuming you have a Microsoft account, follow these steps:

* Go to the Azure Portal, and create a devops instance:

![Create DevOps](/images/azure-create-devops.png)

* Create an organization:

![Create DevOps](/images/azure-create-devops-org.png)

* Complete the form:

![Create DevOps](/images/azure-create-devops-form.png)

Whatever you have chosen, you should now have a Devops at a URL such as https://dev.azure.com/your-name. Start by creating a new project.
